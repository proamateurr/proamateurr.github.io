My <Coding> Cheat Sheet

HTML Document Standards

Tag Name Description
<!doctype HTML> HTML Declaration Declares that the file being used is a HTML document.
<HTML> HTML Anything in between these tags are interpreted as HTML code
<a> Anchor Achors a link to text, has to be used with the 'href' attribute to link a URL or relative path
<!--text--> Comment Makes text a comment inside the HTML document

HTML Tables

Tag/Attribute Name Description
<table> Table This tag creates a table and is the parent to the other table elements
<tbody> Table Body Contains all of the table's data
<thead> Table Head Keeps the headings tags separate from the body content
<tfoot> Table Footer Footer of the table
<tr> Table Row Adds a row to the table
<td> Table Data Adds data to a cell in the table
<th> Table Heading Adds headings to rows and columns,
scope attribute specifies the location of the heading
'colspan' Column Span Allows data to span accross multiple columns
'rowspan' Row Span Allows data to span accorss multiple rows
'scope' Scope Specifies whether a header cell is header for a column, row, or group of columns or rows

Semantic HTML

Tag Name Description
<header> Header Usually contains navigation links of introductory content
<nav> Navigation Used to deine a block of navigation links
<main> Main Encapsulates dominant content within a webpage
<footer> Footer Content at the bottom of the page such as contact info, copyright, etc.
<section> Section Defines elements in document, such as chapters, headings, or any other area of the document with the same theme
<article> Article Holds content that makes sense on its own, such as articles, blogs, quotes, etc.
<aside> Aside Used to make additional information that can enhance another element but isn't required in order to understand the main content
<figure> Figure Used to encapsulate media
<sfigcaption> Figure Caption Used to describe media in <figure> element tag
<audio> Audio Used to embed audio into a document
<video> Video Used to add videos to a website
<embed> Embed Can embed any media content, however it is a deprecated tag
